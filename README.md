# Nancy Drew Mystery Game

A point-and-click mystery game built in Unity for WebGL.

## About

This is a Nancy Drew mystery game being built collaboratively using AI assistance. The game features:

- Point-and-click adventure gameplay
- Mystery storyline with clues and puzzles
- Dialogue system
- Inventory/evidence collection
- Multiple chapters

## Tech Stack

- Unity 6
- C#
- WebGL build target

## Development

This game is being built conversationally using Claude Code and pushed to GitHub. The goal is to minimize direct Unity work after initial setup.
